== Description ==

= Coinsnap Bitcoin Invoice Form =


Coinsnap Bitcoin Invoice Form works with Coinsnap or your own BTCPay Server.

= Requirements: =

* A WordPress website
* The Coinsnap Bitcoin Invoice Form plugin
* A [Coinsnap account](https://app.coinsnap.io/register) or your own BTCPay Server
